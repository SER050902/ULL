# 🟦 Ultraman Local

**Ultraman Local** 是一个基于 **Flask** 的本地轻量级管理面板，用于在浏览器中统一查看和管理本机的常用信息与操作。

> 用最小的依赖，搭建一个清晰、可扩展的本地管理中枢（Local Hub）。

当前版本专注于基础架构与核心功能的搭建，后续将逐步扩展更多实用的小工具与管理能力。

---

## ✨ 功能特性

### 🖥️ 系统信息
- CPU 使用情况
- 内存占用
- 磁盘分区与使用率
- 系统运行时间

### 🌐 网络信息
- 所有网络接口（IPv4 / IPv6）
- 局域网 IP
- 公网 IP（自动获取）

### 📁 本地文件管理器
- 目录浏览
- 文件上传与下载
- 新建文件夹
- 重命名 / 删除（带安全限制）
- 系统目录保护与提示（如 Windows `C:` 根目录）

### 🧱 可扩展架构
- 模块化页面设计
- 清晰的后端接口
- 为后续功能预留扩展空间

---
